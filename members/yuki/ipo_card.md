## IPOカード

コードを書く**前に**、必ずIPOカードを記入する。

### 機能名: おしゃべり挨拶ボット

| 項目        | 内容                                                          |
| ----------- | ------------------------------------------------------------- |
| **Input**   | 名前                                                          |
| **Process** | 受け取った名前に、時間に応じた挨拶をつける ＋名前に応じた挨拶 |
| **Output**  | 名前と挨拶                                                    |

### 機能名: 割り勘計算機

| 項目        | 内容                                           |
| ----------- | ---------------------------------------------- |
| **Input**   | 合計金額と人数                                 |
| **Process** | 受け取った合計金額を＋消費税を加算、人数で割る |
| **Output**  | 一人あたりの金額                               |

- ToDoリスト: 表示だけする機能をつける
